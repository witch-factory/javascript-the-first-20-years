# C. 용어사전

<span id='actionscript'>
  **ActionScript** 명사. ECMAScript의 방언으로 Adobe Flash 플랫폼에서 사용되는
  프로그래밍 언어.
</span>

<span id='attribute'>
  **attribute** 명사. 1. ECMAScript 명세에서 객체 프로퍼티의 설정 가능한 특성. 2. HTML에서, 여는 태그 내의 동작 수정자. `<a>`태그의 `href`와 같은 것이다.
</span>

<span id='awk'>
  **AWK** 명사. Unix용으로 처음 만들어진 도메인 특화 텍스트 처리 언어 [Aho et
  al. 1988](/docs/appendix/references#aho-1988).
</span>

<span id='binding'>
  **binding** 명사. 이름을 변수 또는 상수 값에 매핑하는 연결.
</span>

<span id='breaking-change'>
  **breaking-change** 명사. 기존 프로그램이 거부되거나 제대로 작동하지 않게
  만드는 프로그래밍 언어나 플랫폼의 변경사항
</span>

<span id='browser-wars'>
  **browser wars** 명사. 브라우저 공급 업체들이 시장 지배를 위해 치열하게
  경쟁했던 기간.
</span>

<span id='chrome'>
  **Chrome** 명사. Google에 의해 개발되고 배포된 웹 브라우저.
</span>

<span id='chromium'>**Chromium** 명사. Chrome 브라우저의 오픈 소스 코어.</span>

<span id='class'>
  **class** 명사. 프로그래밍 언어 개념으로, 객체 그룹이 공유하는 공통
  인터페이스와 구현을 정의하는 메커니즘.
</span>

<span id='classical-inheritance'>
  **classical inheritance** 명사. 객체가 클래스 정의 체인으로부터 상태와 동작을
  획득하게 되는 상속 메커니즘.
</span>

<span id='coffeescript'>
  **CoffeeScript** 명사. JavaScript로 컴파일되는 프로그래밍 언어로 Jeremy
  Ashkenas에 의해 만들어졌다.
</span>

<span id='commonjs'>
  **CommonJS** 명사. 브라우저 이외의 환경에서도 JavaScript 기술을 사용할 수
  있도록 하기 위해 Kevin Dangoor에 의해 시작된 프로젝트.
</span>

<span id='compiler'>
  **compiler** 명사. 프로그램을 (일반적으로) 프로세서에서 직접 실행될 수 있는
  기계 언어로 변환하는 엔진.
</span>

<span id='constructor'>
  **constructor** 명사. constructor function 항목 참조.
</span>

<span id='constructor-function'>
  **constructor function** 명사. 객체를 할당하고 초기화하는 JavaScript 함수로,
  `new` 연산자를 사용하여 호출될 수 있음.
</span>

<span id='cyclic-garbage-collection'>
  **cyclic garbage collection** 동사. 다른 곳에서는 도달할 수 없는 고립된 순환
  구조에 할당된 공간을 회수할 수 있는 메모리 관리 과정.
</span>

<span id='dart'>
  **Dart** 명사. 원래는 웹 브라우저에서 JavaScript를 대체할 목적으로 Google에
  의해 개발된 클래스 기반 객체 지향 프로그래밍 언어.
</span>

<span id='declarative'>
  **declarative** 형용사. 원하는 결과의 특성을 기술하는 것에 기반한 프로그래밍
  접근 방식.
</span>

<span id='delegation'>
  **delegation** 명사. 객체가 클래스 정의가 아닌 다른 객체로부터 상태와 동작의
  일부 또는 전부를 획득하는 메커니즘.
</span>

<span id='discriminated-union'>
  **discriminated union** 명사. 내부에 여러 대체 가능한 구조들이 들어 있는
  데이터 레코드이며 실제 구조는 명시적인 태그 값을 통해서 정해진다.
</span>

<span id='destructuring'>
  **destructuring** 명사. 배열이나 객체의 프로퍼티를 배열이나 객체 리터럴과
  유사한 문법을 사용하여 참조하는 것.
</span>

<span id='desugar'>
  **desugar** 동사. 프로그래밍 언어의 문장이나 연산을 더 기본적인 문장과
  연산으로 분해하는 것.
</span>

<span id='devdiv'>**DevDiv** 명사. Microsoft의 개발자 도구 부서.</span>

<span id='dynamic-language'>
  **dynamic language** 명사. 실행 이전 시점에 프로그램의 분석을 거의 또는 전혀
  요구하지 않는 프로그래밍 언어. 언어에서 지정한 대부분의 오류 검사는 실행 중에
  발생하며, 일반적으로 동적 언어로 짜인 프로그램은 런타임에 구성되거나 수정될 수
  있음; cf. _static language_.
</span>

<span id='dynamically-typed'>
  **dynamically typed** 형용사. 데이터 타입의 안전성 제약이 주로 프로그램 실행
  시점에 검사되는 프로그래밍 언어.
</span>

<span id='es-discuss'>
  **es-discuss** 명사. ECMAScript의 발전을 논의하기 위한 공개 이메일 포럼.
</span>

<span id='ecma-262'>**ECMA-262** 명사. ECMAScript 언어 명세.</span>

<span id='ecma-402'>**ECMA-402** 명사. ECMAScript 초기화 API 명세.</span>

<span id='engine'>**engine** 명사. 프로그램을 실행하는 메커니즘.</span>

<span id='es4-discuss'>
  **es4-discuss** 명사. `es-discuss` 이메일 포럼의 원래 이름.
</span>

<span id='es-next'>
  **ES.next** 명사. TC39 내에서 ECMA-262의 다음 버전을 가리키는 데에 때때로
  사용된 용어.
</span>

<span id='exotic-object'>
  **exotic object** 명사. 모든 객체가 지원해야 하는 필수 내부 메서드 중 하나
  이상이 없거나 기본 동작과 다른 방식으로 동작하는 JavaScript 객체; cf. ordinary
  object.
</span>

<span id='expando-property'>
  **expando property** 명사. 객체 생성 후 객체에 동적으로 추가된 속성.
</span>

<span id='factory-function'>
  **factory function** 명사. 새 객체를 생성해 반환하는 함수.
</span>

<span id='firefox'>
  **Firefox** 명사. Mozilla에 의해 개발되고 배포된 웹 브라우저.
</span>

<span id='first-class'>
  **first-class** 형용사. 데이터 값으로 사용될 수 있는 프로그래밍 언어 런타임
  엔티티; 예를 들어, 변수에 할당되거나 함수 인자로 사용되거나, 함수에서
  반환되거나, 자료구조에 저장될 수 있음.
</span>

<span id='flash'>
  **Flash** 명사. 리치 인터넷 애플리케이션 및 기타 용도를 지원하는 Adobe의
  멀티미디어 소프트웨어 플랫폼.
</span>

<span id='free-variable'>
  **free variable** 명사. 참조할 수 있지만 로컬 스코프에서 정의되지 않은
  바인딩의 변수
</span>

<span id='function'>
  **function** 명사. 하위 프로그램(subroutine); 프로그램의 매개변수화된 부분.
</span>

<span id='hackathon'>
  **hackathon** 명사. 프로그래머들이 몇 일 동안 모여 프로젝트 협업을 하는
  이벤트.
</span>

<span id='harmony'>
  **Harmony** 명사. ES4<sub>2</sub>를 포기한 후 ECMA-262 개발을 위한 TC39의
  코드명.
</span>

<span id='host-object'>
  **host object** 명사. Javascript 엔진에 의해 제공되는 객체 또는 객체 클래스로
  호스트 애플리케이션 또는 플랫폼의 기능에 대한 접근을 제공함.
</span>

<span id='imperative'>
  **imperative** 형용사. 원하는 결과를 얻기 위해 수행될 일련의 단계들을 기술하는
  계산 접근 방식.
</span>

<span id='inherit'>
  **inherit** 동사. 객체 지향 언어에서, 간접적으로 특성을 획득함.
</span>

<span id='inheritance'>
  **inheritance** 명사. 객체 지향 언어에서, 객체가 일부 또는 전부의 데이터와
  행동을 상속하는 메커니즘.
</span>

<span id='inherited-property'>
  **inherited property** 명사. 프로토타입으로부터 상속된 JavaScript 객체의 속성.
</span>

<span id='internal-method'>
  **internal method** 명사. 객체의 시맨틱을 정의하는 메커니즘으로 사용자가 접근
  가능한 부분이 아니다.
</span>

<span id='internal-property'>
  **internal property** 명사. 객체 시맨틱의 일부를 정의하는데 필요한 상태를
  저장하기 위해 내부 메서드가 사용하는 객체의 부분.
</span>

<span id='interpreter'>
  **interpreter** 명사. 프로그램의 표현을 훑어 나가면서 통해 각 연산을 만나는
  대로 수행함으로써 프로그램을 실행하는 엔진.
</span>

<span id='internationalization'>
  **internationalization** 명사. 프로그램이 다양한 인간 언어, 문자, 작성 규칙을
  처리할 수 있도록 하는 과정.
</span>

<span id='internet-explorer'>
  **Internet Explorer** 명사. Microsoft에 의해 개발되고 배포된 웹 브라우저.
</span>

<span id='java'>
  **Java** 명사. Sun Microsystems에 의해 개발된 클래스 기반 객체 지향 프로그래밍
  언어.
</span>

<span id='javascript-engine'>
  **JavaScript engine** 명사. JavaScript 언어의 구현체.
</span>

<span id='jscript'>
  **JScript** 명사. Microsoft에 의해 구현된 JavaScript의 방언.
</span>

<span id='lambda-expression'>
  **lambda expression** 명사. 식별자에 바인딩되지 않은 함수 정의. 받는 인자와
  그걸 이용한 실행 또는 평가의 단계들을 정의하는 표현식이다. 람다 대수와
  Lisp에서 유래함.
</span>

<span id='leaky-abstraction'>
  **leaky abstraction** 명사. 숨겨지거나 비공개여야 할 추상화의 세부 정보를
  의도치 않게 드러내는 추상화.
</span>

<span id='mashup'>
  **mashup** 명사. 독립적으로 관리되는 서버에서 각각 도착하는 JavaScript 코드와
  콘텐츠를 동적으로 결합한 웹 페이지.
</span>

<span id='membrane'>
  **membrane** 명사. 객체-능력(object-capability) 시스템에서 객체를 변조 방지한
  상태로 보안 컨텍스트 간에 공유하기 위해 사용되는 메커니즘.
</span>

<span id='metaobject-protocol'>
  **metaobject protocol** 명사. 객체 지향 언어에서, 언어에서의 기본적인 동작을
  정의하고 접근하기 위한 잘 지정된 인터페이스.
</span>

<span id='method'>**method** 명사. 객체의 구성 요소인 함수.</span>

<span id='netscape-navigator'>
  **Netscape Navigator** 명사. Netscape Communications에 의해 개발되고 배포된 웹
  브라우저.
</span>

<span id='mocha'>
  **Mocha** 명사. JavaScript가 되었던 언어의 코드명; 또한, Netscape의 원래
  JavaScript 엔진의 이름.
</span>

<span id='mosaic'>
  **Mosaic** 명사. NCSA에서 Marc Andreessen과 Eric Bina에 의해 개발된 웹
  브라우저.
</span>

<span id='nodejs'>
  **Node.js** 명사. 2009년 Ryan Dahl에 의해 처음 개발된 JavaScript 기반 서버
  플랫폼.
</span>

<span id='nominal-type-system'>
  **nominal type system** 명사. 각 타입 정의가 고유한 데이터 타입을 도입하는
  타입 시스템; 일부 객체 지향 언어에서는 클래스 정의가 명목 타입 정의로 취급됨.
</span>

<span id='non-normative'>
  **non-normative** 형용사. 어떤 요구 사항도 정의하지 않는 표준 문서의 부분.
</span>

<span id='no-op'>**no-op** 명사. 아무 것도 하지 않는 연산.</span>

<span id='normative'>
  **normative** 형용사. 요구 사항을 정의하는 표준 문서의 부분.
</span>

<span id='object'>
  **object** 명사. 데이터와 동작을 일급 복합 엔티티로 그룹화하는 연산 장치;
  객체를 정의하고 조작하는 메커니즘은 프로그래밍 언어마다 다름.
</span>

<span id='opera'>
  **Opera** 명사. Opera Software에 의해 개발되고 배포된 웹 브라우저.
</span>

<span id='own-property'>
  **own property** 명사. 상속된 것이 아니라 객체에 직접 속해 있는 JavaScript
  객체의 속성.
</span>

<span id='ordinary-object'>
  **ordinary object** 명사. 모든 객체가 지원해야 하는 필수 내부 메서드에 대한
  기본 동작을 가진 JavaScript 객체; cf. exotic object.
</span>

<span id='profile'>
  **profile** 명사. 특정 장치, 플랫폼 또는 애플리케이션의 요구 사항에 맞춰
  조정된 기능의 집합.
</span>

<span id='polyfill'>
  **polyfill** 명사. 브라우저에서 제공해야 하지만 버전 차이 등의 이유로 지원하지
  않는 API를 제공하는 라이브러리.
</span>

<span id='proper-tail-call'>
  **proper tail call** 명사. 호출한 함수로 제어를 전혀 반환하지 않는 꼬리 호출.
</span>

<span id='property'>**property** 명사. JavaScript 객체의 구성 요소.</span>

<span id='property-key'>
  **property key** 명사. 객체의 특정 프로퍼티를 식별하는 데 사용되는 문자열 또는
  심볼.
</span>

<span id='prototype'>
  **prototype** 명사. 다른 객체에 상속된 상태와 동작을 제공하는 객체.
</span>

<span id='prototypal-inheritance'>
  **prototypal inheritance** 명사. 객체가 프로토타입 체인에서 상태와 동작 일부
  또는 전부를 획득하는 상속 메커니즘.
</span>

<span id='safari'>**Safari** 명사. Apple에 의해 개발된 웹 브라우저.</span>

<span id='sandbox'>
  **sandbox** 명사. 호스트 환경과 다른 프로그램이 직접 데이터에 접근하거나
  간섭하는 것을 방지하기 위해 프로그램이나 프로그램의 일부를 분리하는 메커니즘.
</span>

<span id='secure-ecmascript'>
  **Secure ECMAScript** 명사. 보안 취약점이 될 수 있는 기능을 제거한 ECMAScript
  방언.
</span>

<span id='self-hosting'>
  **self-hosting** 명사. 프로그래밍 언어 엔진 코드의 일부를 해당 엔진에서
  구현하는 언어로 작성하는 것.
</span>

<span id='shadow'>
  **shadow** 동사. 상속된 특성을 재정의하지 않고 덮어쓰기.
</span>

<span id='silverlight'>
  **Silverlight** 명사. 다채로운 인터넷 애플리케이션을 위한 Microsoft 플랫폼.
</span>

<span id='scope'>
  **scope** 명사. 변수(또는 선언된 바인딩)가 참조될 수 있는 프로그램의 영역.
</span>

<span id='scope-contour'>
  **scope contour** 명사. 중첩된 스코프 그룹 내에서 단일 스코프의 표현.
</span>

<span id='scripting-language'>
  **scripting language** 명사. 컴퓨팅 시스템, 애플리케이션 또는 다른 언어로
  정의된 연산 추상화 작업을 조율하는 데 쓰이는 프로그래밍 언어로 일반적으로
  간단한 편이다.
</span>

<span id='spidermonkey'>
  **SpiderMonkey** 명사. 1996년 이후 개발된 모든 Netscape 및 Mozilla
  브라우저에서 사용되는 JavaScript 엔진.
</span>

<span id='static-language'>
  **static language** 명사. 실행 이전 시점에 프로그램의 일부 또는 상당 부분을
  분석해야 하는 프로그래밍 언어. 언어에서 규정한 대부분의 오류 검사는 실행
  이전에 발생하고 일반적으로 런타임에는 프로그램이 수정될 수 없다. cf. _dynamic
  language_.
</span>

<span id='statically-typed'>
  **statically typed** 형용사. 데이터 타입의 안전성 제약에 대한 집행이 주로
  프로그램 실행 이전에 수행되는 프로그래밍 언어.
</span>

<span id='tail-call'>
  **tail call** 명사. 메서드 내에서 메서드나 함수 호출을 하는 것이 메서드의 최종
  동작이 되는 것. 이러한 호출의 구현은 메서드로의 제어 반환을 포함할 수도 있으나
  필수적인 것은 아님. cf. proper tail call.
</span>

<span id='transpiler'>
  **transpiler** 명사. 하나의 언어로 작성된 프로그램을 다른 언어로 컴파일하는
  언어 프로세서.
</span>

<span id='type'>
  **type** 명사. 표현과 사용 가능한 연산 등의 공통 특성을 공유하는 값의 범주.
</span>

<span id='type-annotation'>
  **type annotation** 명사. 변수나 다른 바인딩에 타입을 연결하는 데 사용되는
  문법적인 형태.
</span>

<span id='url'>
  **URL** 명사. 월드 와이드 웹 페이지의 주소. Uniform Resource Locator의 약자.
</span>

<span id='value'>
  **value** 명사. 프로그램이 조작하는 정보의 단위; 타입이 지정된 프로그래밍
  언어에서는 값이 다양한 타입으로 분류됨.
</span>

<span id='v8'>**V8** 명사. Chrome 브라우저에서 사용되는 JavaScript 엔진.</span>

<span id='webkit'>
  **WebKit** 명사. Apple Safari 및 일부 다른 브라우저에서 사용되는 오픈 소스
  브라우저 코어.
</span>

<span id='web-2.0'>
  **Web 2.0** 명사. 사용자 정의 콘텐츠에 초점을 맞춘 웹 애플리케이션 스타일;
  대부분 사용자와 활발하게 상호작용하며 AJAX 기술을 사용하여 구축됨.
</span>

<span id='web-reality'>
  **Web Reality** 명사. 현재 존재하는 월드 와이드 웹의 기술적인 면과 특성. 또한
  현존하는 웹 페이지와 애플리케이션에 의해 사용될 때의 기술적인 부분과 특성. 웹
  인프라 구조의 확장은 일반적으로 이러한 기존의 기술적 측면 및 특성을 그대로
  유지할 수 있게 해야 함.
</span>
